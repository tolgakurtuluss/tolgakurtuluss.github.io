EtihadAirlinesGPTPlugin-langchain is a Python script that uses LangChain to create an intelligent agent capable of interacting with Etihad Airways' official GPT Plugin for flight searches and bookings. The agent allows you to query flights in natural language (e.g., "Find flights from Istanbul to Abu Dhabi on August 1, 2023"), and it retrieves real-time flight options—including departure/arrival times, duration, stops, prices in AED, and direct booking links via the Etihad GPT Plugin.

Key Features:
- Natural language flight search (origin, destination, date)
- Integration with Etihad's GPT Plugin (hosted at https://gpt-etihad.botim.me)
- Detailed results with multiple flight options and actionable booking URLs
- Lightweight implementation using only LangChain and OpenAI
